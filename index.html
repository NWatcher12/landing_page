HeartStone Academy

Heartstone Academy is a group passion project under the UC Irvine Video Game Development Club for the 2024 Spring submission. Under the leadership of Ziqing Li, I worked within the team with fellow undergraduate students to create an intruiging game with appealing visuals, audio, and story. Being one of the only 2 programmers wtihin the team posed a challenge: large workloads while juggling between works and school. Despite the challenges, we manage to create finish the demo of the game in around 7 weeks.

The game is created using the Ren'Py programming language--a functional programming language made in Python. Using the program, we are able to integrate various media: visual, textual, and audio.

Despite the challenges that we faced as a team, we manage to prevails. In the future, it is possible to experiment with intergration of Live2D--an extension that will allow animation of 2D animation--to improve user experience. It is possible to also create records of images that the user had encounter in game.
